# TP2 : Entités Salle et Utilisateur avec Hibernate
# Objectifs :
> Créer un projet Maven avec Hibernate et H2   
> Configurer la génération automatique du schéma avec hibernate.hbm2ddl  
> Créer deux entités (Salle et Utilisateur) avec des validations  
> Implémenter des services pour les opérations CRUD  
> Tester les opérations CRUD via une application principale et des tests unitaires
